# Installation & Setup

1. Clone the repo and install all the dependencies
   ```
   npm install
   ```
2. It's an Expo managed project so you'll need <b>expo-cli</b> installed globally
   ```
   npm install -g expo-cli
   ```
3. Start the dev server
   ```
   expo start
   ```
